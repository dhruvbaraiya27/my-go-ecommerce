# API Documentation

## Authentication Endpoints
- POST /api/register
- POST /api/login
- POST /api/logout

## User Endpoints
- GET /api/users/profile
- PUT /api/users/profile
- DELETE /api/users/profile

## Product Endpoints
- GET /api/products
- GET /api/products/:id
- POST /api/products (Admin)
- PUT /api/products/:id (Admin)
- DELETE /api/products/:id (Admin)

## Order Endpoints
- GET /api/orders
- GET /api/orders/:id
- POST /api/orders
- PUT /api/orders/:id/status (Admin)
